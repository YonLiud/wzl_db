# Weazel MDC

## About
Database managment application built for Weazel News in VitalRP.

## Installation & Runtime

#### Requirements:
1) Python 3.6+
2) Flask
3) SQLite3
4) Git

#### How To Install

Clone the project
```sh
git clone https://github.com/YonLiud/wzl_db
```

After cloning, you can run the application by running the following command:

Linux: 

```sh
python3 run.py
```

Windows:
```cmd
py run.py
```
